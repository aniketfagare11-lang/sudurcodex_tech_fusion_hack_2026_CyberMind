# 📌 Project Overview

**GameConnect** is a web platform designed to help gamers find teammates, join tournaments, and compete with skilled players.

Many esports players struggle to find reliable teammates for ranked matches or tournaments. GameConnect solves this problem by allowing players to discover teammates based on their game, rank, and playstyle.

The platform creates a **gaming ecosystem** where players can:

* Find teammates instantly
* Join esports tournaments
* Track leaderboard rankings
* Manage their gaming profile

---

# ✨ Key Features

### 👥 Find Teammates

Players can search and filter gamers based on:

* Game
* Rank
* Playstyle
* Online status

Players can also send **squad invites** directly.

---

### 🏆 Tournaments System

Users can browse esports tournaments with details such as:

* Entry fee
* Prize pool
* Available slots
* Match dates
* Game type

Players can join tournaments with a simple **Join Tournament** button.

---

### 📊 Leaderboard

The leaderboard shows top players based on:

* Wins
* Points
* Rank tier

This helps competitive players track their progress.

---

### 👤 Player Profile

Players can maintain their gaming profile including:

* Display name
* Player ID
* Game rank
* Playstyle
* Region
* Game statistics

---

### 📈 Dashboard

The dashboard shows player statistics such as:

* Matches played
* Wins
* K/D ratio
* Rank points

# ⚙️ How the Project Works

1. User opens the GameConnect website.
2. They can explore available games on the homepage.
3. Players can search for teammates in the **Find Teammates** section.
4. Players can invite other gamers to join their squad.
5. Users can browse available tournaments.
6. Players can join tournaments directly.
7. Leaderboards display top performing players.

# 🧩 Problems Faced During Development

### 1️⃣ Deployment Issues

While deploying the project on **Vercel**, the website showed a **404 error**.

**Reason:**
The project structure was not properly configured for static deployment.

**Solution:**
The project was deployed using **Netlify**, which supports static HTML/CSS/JS projects easily.

---

### 2️⃣ JavaScript File Loading Error

During development, some JavaScript functions were not working.

**Reason:**
Incorrect file paths caused the browser to fail loading `script.js`.

**Solution:**
The script paths were corrected and properly linked at the bottom of the HTML file.

---

### 3️⃣ UI Layout Issues

Some UI components such as:

* tournament cards
* prize badges
* player cards

were not aligned correctly.

**Solution:**
Custom CSS styling and responsive layout adjustments were implemented.

---

### 4️⃣ Button Actions Not Working

Some interactive buttons like:

* Join Tournament
* Invite Player

did not trigger actions initially.

**Solution:**
JavaScript event functions were added and tested to ensure proper interaction.
